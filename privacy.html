Privacy Policy — NemoChat

Effective: April 23, 2026

NemoChat is a local-only application. We (the developer) do not collect, store, transmit, or process any personal data on any server.

1. Data Collection

NemoChat collects no data. There is no account system, no analytics SDK, no telemetry, and no tracking pixel of any kind.

2. Data Storage

All conversation history, attached images, and settings are stored locally on your device using Apple's SwiftData framework. API keys and access tokens you provide (Hugging Face access token, Brave Search API key) are stored in the iOS Keychain and remain on your device. Nothing is synced to iCloud, our servers, or any third party. Uninstalling the app removes this data.

3. Permissions

When a feature requires access (camera, calendar, contacts, location, reminders), iOS prompts you. These permissions are used only for the immediate tool action and are never recorded or transmitted.

4. Third-Party Services

Some optional features send network requests directly from your device to third parties. When you use these, the third party will necessarily see your IP address (as with any HTTPS request) in addition to the data listed below. We (the developer) do not receive, proxy, or log any of this traffic.

  • Hugging Face — used for the one-time model download and any subsequent model downloads you initiate. Hugging Face receives your IP address and the model file path you request. If you provide a Hugging Face access token, it is sent only to Hugging Face to authenticate the download. See Hugging Face's privacy policy.
  • Open-Meteo — contacted only when the weather tool is invoked. Open-Meteo receives your IP address and the coordinates used for the forecast (either your device location, with your permission, or a location you specify). No API key and no account are used. See Open-Meteo's terms and privacy notice.
  • Brave Search — contacted only if you add a Brave Search API key and the web search tool is invoked. Brave receives your IP address, your search query, and your API key. See Brave's privacy policy.

5. Children's Privacy

NemoChat does not knowingly collect information from anyone, including children under 13.
